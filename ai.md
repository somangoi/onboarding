- 사용 위치

  - src/entities/onboarding/model/reducer.test.ts
- 사용 이유
  - 테스트 코드 작성을 빠르고 효율적으로 하기 위해
- 핵심 프롬프트
  ```
  @{파일명} 파일을 참고하여 {목적} 테스트 코드를 작성.
  ```
- 활용방식

